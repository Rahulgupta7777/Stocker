# Stocker

Stocker is a stock market analysis and portfolio management platform designed to streamline market data tracking, portfolio monitoring, and basic strategy experimentation. The system features a secure client–server architecture and a well-structured Prisma–PostgreSQL database schema, enabling reliable data handling, scalable analytics workflows, and responsive user interfaces.

The platform supports market data ingestion, portfolio and position management, alert automation, and lightweight backtesting, providing a practical foundation for traders and developers to prototype strategies and visualize performance efficiently.

## Features
- **Market Data Tracking**: Real-time and historical data ingestion.
- **Portfolio Monitoring**: Track positions and performance.
- **Strategy Experimentation**: Prototype and test trading strategies.
- **Secure Architecture**: Client-server model with Google Authentication (OAuth2).
- **Alert Automation**: Automated notifications for market events.

## Tech Stack
- **Languages**: TypeScript, JavaScript, Python
- **Frontend**: React, TailwindCSS
- **Backend**: Node.js, Prisma ORM, PostgreSQL
- **Analysis Engine**: Python (Pandas, TA-Lib)
- **Auth**: Google Authentication (OAuth2)
